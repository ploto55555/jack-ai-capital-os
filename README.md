# Jack AI Capital OS

Personal trading operating system prototype for discipline, risk control, journal, mistake memory and Abu-style setup review.

## V1 Scope

- Pure HTML/CSS/JavaScript
- DOS terminal style UI
- Mock market decisions
- Command input
- Watchlist click interaction
- Risk calculator mock
- No broker connection
- No auto-trading
- No real-time data yet

## Files

```text
index.html
style.css
script.js
README.md
```

This system is for personal decision support only. It does not execute trades automatically.
